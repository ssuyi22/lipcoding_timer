## 🎮 핵심 기능 요약

### 1️⃣ 타이머 (Timer)
```
세 가지 모드:
├─ 집중 (Focus): 25분
├─ 짧은 휴식 (Short Break): 5분
└─ 긴 휴식 (Long Break): 15분

시각화:
├─ 원형 진행도 (Ring)
├─ MM:SS 형식 디스플레이
└─ 1시간 이상 시 시간 배지
```

### 2️⃣ 통계 (Statistics)
```
일일 통계 (오늘):
├─ 완료한 집중 세션 수
├─ 연속 완료 (Streak)
└─ 총 집중 시간(분)

누적 통계 (예측 기록):
├─ 예측 기록 테이블 (최대 50개)
└─ 카테고리별 분석
```

### 3️⃣ 예측 모드 (Prediction Mode) - Premium
```
입력:
├─ 작업명 + 카테고리
├─ 예상 소요 시간
└─ 예상 방해 횟수

출력:
├─ 실제 소요 시간 기록
├─ 차이 계산
└─ AI 분석 (Copilot)
```

### 4️⃣ 인증 (Authentication)
```
등록/로그인:
├─ 이메일 + 비밀번호
├─ 토큰 기반 인증
└─ localStorage에 저장

사용자 계획:
├─ 무료 (Free): 기본 타이머 + 통계
└─ 유료 (Premium): 모든 기능 + AI 분석
```

### 5️⃣ AI 분석 (Copilot Integration) - Premium
```
GitHub 토큰 설정:
├─ 브라우저 localStorage에만 저장
├─ 서버에는 전송 안함
└─ 수동으로 관리

분석 내용:
├─ 작업 패턴 분석
├─ 시간 추정 편향 감지
└─ 개선 제안
```

## 📊 모든 테이블 한눈에 보기

### 🔹 오늘 통계 (Today Statistics)
```
┌─────────────────────────────────────┐
│ 완료한 집중 세션 │ 연속 완료 │ 총 집중 시간  │
│       0        │    0    │     0(분)     │
└─────────────────────────────────────┘
```
**위치**: 타이머 우측 상단
**데이터 저장**: sessionStorage (일별 초기화)
**갱신**: 타이머 완료 시 실시간

### 🔹 예측 기록 (Prediction History)
```
┌────────────────────────────────────────────────────────┐
│ 작업명 │ 카테고리 │ 예측 │ 실제 │ 차이 │ AI분석 │
├────────────────────────────────────────────────────────┤
│ 알고리즘 │ 코딩 │ 15분 │ 1분 │ -14분 │ 규칙 │
│ 기획서 │ 문서 │ 120분 │ 1분 │ -119분 │ 규칙 │
└────────────────────────────────────────────────────────┘
```
**위치**: 메인 페이지 하단
**저장 제한**: 최대 50개 기록
**데이터 저장**: localStorage

### 🔹 카테고리별 통계 (Category Statistics)
```
┌──────────────────────────────────────┐
│ 카테고리 │ 회수 │ 평균오차 │ 정확도 │
├──────────────────────────────────────┤
│ 문서작성 │  1  │ -119.0분 │ -376% │
│  코딩    │  2  │ -19.0분  │  24% │
│ 미분류   │  2  │ -24.0분  │  4%  │
└──────────────────────────────────────┘
```
**위치**: 예측 기록 테이블 하단
**계산**: 예측 기록에서 자동 집계

### 🔘 버튼 (140px 표준 너비)
| 버튼 | 위치 | 기능 | 스타일 |
|------|------|------|--------|

| 리셋 | 우측 | 모드 초기화 | Ghost (흰색) |
| 건너뛰기 | 우측 | 다음 모드로 | Ghost (흰색) |
| 적용 | 우측 | 커스텀 시간 설정 | Ghost (흰색) |
| 자동 연결 | 중단 | 컨디션 자동 연결 | Ghost |
| 자동추천 적용 | 중단 | 루틴 적용 | Ghost |
| 예측 세션 시작 | 중단 | 예측 시작 | Primary |
| 예측 세션 종료/분석 | 중단 | 예측 종료 | Ghost |
| 📥 CSV 내보내기 | 하단 | 예측 기록 내보내기 | Ghost |

### 📝 입력 필드 (Number, Text, Range, Select)

#### 타이머 관련
- **현재 모드 시간(분)**: number, 1~180, 기본값 25
- **예상 소요 시간(분)**: number, 1~300, 기본값 25
- **예상 방해 횟수**: number, 0~20, 기본값 0

#### 작업 관련
- **작업 이름**: text, 임의의 텍스트
- **카테고리**: text, 임의의 텍스트 (예: "코딩", "문서작성")

#### 개인 컨디션
- **수면 시간(시간)**: number, 3~12, 스텝 0.5, 기본값 7
- **컨디션 점수(1~5)**: range, 1~5, 기본값 3 (중간)
  - 1 = 매우 피곤
  - 3 = 중간
  - 5 = 최고조
- **외부 일정 강도**: select, 여유/보통/바쁨, 기본값 보통

#### 토큰 관리
- **GitHub Personal Access Token**: password, 숨김

### 📊 체크박스
- **Copilot SDK AI 분석 사용**: 기본값 checked

## 🔄 주요 동작 흐름

### 📌 타이머 작동 흐름
```
[시작 클릭]
  ↓
setInterval 시작
  ↓
매초 tick()
  ↓
remaining-- → render() 호출
  ↓
CSS 변수 업데이트 (--ring-angle)
  ↓
MM:SS 업데이트
  ↓
remaining == 0 ?
  ├─ YES → 세션 완료 → 통계 업데이트
  └─ NO → 계속 진행
```

### 📌 예측 기록 저장 흐름
```
[작업 입력]
  ↓
[예측 세션 시작]
  ↓
predictionSession 객체 생성
  ↓
[실제 작업 진행]
  ↓
[예측 세션 종료/분석]
  ↓
실제 소요 시간 계산
  ↓
차이 = 실제 - 예측
  ↓
AI 분석 요청 (Copilot SDK)
  ↓
결과 저장 → localStorage
  ↓
테이블 렌더링
```

### 📌 인증 흐름
```
[로그인 클릭]
  ↓
인증 모달 표시
  ↓
이메일 + 비밀번호 입력
  ↓
[로그인 제출]
  ↓
API 요청 (POST /api/login)
  ↓
성공?
├─ YES → 토큰 저장 → Premium 기능 활성화
└─ NO → 에러 메시지
```
