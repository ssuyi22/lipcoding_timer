# Pomodoro Focus Timer - 기능 문서

## 📋 프로젝트 개요
- **프로젝트명**: Pomodoro Timer (Copilot SDK 통합)
- **버전**: 0.1.0
- **언어**: JavaScript (ES Module), HTML5, CSS3
- **서버**: Node.js + Express
- **인증**: 이메일 기반 로그인/회원가입
- **주요 프레임워크**: GitHub Copilot SDK

---

## 🎯 핵심 기능 (Implemented Features)

### 1. 타이머 모드 시스템
| 모드 | 시간 | 설명 |
|------|------|------|
| **집중 (Focus)** | 25분 | 메인 작업 시간 |
| **짧은 휴식 (Short Break)** | 5분 | 집중 세션 후 짧은 휴식 |
| **긴 휴식 (Long Break)** | 15분 | 4회 집중 후 긴 휴식 |

**구현 위치**: `assets/app.js` (lines 1-5)
```javascript
const MODES = {
  focus: 25 * 60,           // 1500초
  shortBreak: 5 * 60,       // 300초
  longBreak: 15 * 60,       // 900초
};
```

---

### 2. 시각적 타이머 환(Ring) 시각화
- **시작점**: 12시 방향 (60 마크 위치)
- **계산식**: `(remaining / 3600) × 360°`
- **표시 방식**: CSS conic-gradient 이용
- **색상**: 
  - 활성: #ea4335 (진빨강)
  - 배경: #f3d2ce (연한 분홍)
- **규모**: 60분 기준 원형 진행도

**CSS 구현** (`assets/styles.css`):
```css
.clock-rim {
  background: conic-gradient(from 0deg, 
    var(--ring-color, #ea4335) 0deg, 
    var(--ring-color, #ea4335) var(--ring-angle, 360deg), 
    var(--ring-track-color, #f3d2ce) var(--ring-angle, 360deg), 
    var(--ring-track-color, #f3d2ce) 360deg);
}
```

---

### 3. 타이머 컨트롤 버튼
| 버튼 | 기능 | 상태 변화 |
|------|------|---------|
| **시작/일시정지** | 타이머 시작 또는 일시정지 | "시작" ↔ "일시정지" |
| **리셋** | 현재 모드 시간으로 복원 | 남은 시간 → 모드 기본값 |
| **건너뛰기** | 현재 모드 건너뛰고 다음 모드로 | Mode 전환 |
| **적용** | 커스텀 시간 설정 및 적용 | 입력값으로 타이머 초기화 |

**Button 너비**: 140px (표준화)
**최소 높이**: 48px
**간격**: 12px

---

### 4. 일일 통계 대시보드
#### 오늘 통계 테이블

| 지표 | 초기값 | 설명 | HTML ID |
|------|--------|------|---------|
| **완료한 집중 세션** | 0 | 성공적으로 완료한 포모도로 세션 수 | `focusCount` |
| **연속 완료** | 0 | 연속으로 완료한 세션 수 (끊기면 리셋) | `streakCount` |
| **총 집중 시간(분)** | 0 | 완료한 모든 집중 세션의 합계 시간 | `totalFocusMinutes` |

**표시 위치**: "오늘 통계" 카드 (3열 그리드)
**데이터 저장**: sessionStorage (일별로 초기화)

### 5. 인증 시스템
#### 로그인/회원가입
- **인증 방식**: 이메일 + 비밀번호
- **저장 위치**: 브라우저 localStorage
- **토큰 키**: `pomodoro_auth_token_v1`, `pomodoro_auth_state_v1`
- **상태 표시**: "현재 상태" 영역에 표시

#### 접근 제어
| 콘텐츠 | 무료 사용자 | 유료 사용자 |
|--------|-----------|-----------|
| 기본 타이머 | ✓ | ✓ |
| 일일 통계 | ✓ | ✓ |
| 앱 정의 (Pro 오버레이 표시) | ✗ | ✓ |
| 예측 결과 모드 | ✗ | ✓ |
| 컨디션 정보 | ✗ | ✓ |

**인증 UI 요소**:
- 로그인 버튼 (헤더)
- 회원가입 버튼 (헤더)
- 로그아웃 버튼 (로그인 후 표시)
- 인증 모달 팝업

### 6. 예측 결과 모드 (Premium)
#### 입력 필드

| 필드 | 타입 | 범위 | 기본값 | 설명 |
|------|------|------|--------|------|
| **작업 이름** | text | - | "" | 수행할 작업의 이름 |
| **카테고리** | text | - | "" | 작업 분류 (예: 문서 작성, 코딩) |
| **예상 소요 시간(분)** | number | 1~300 | 25 | 예상 작업 시간 |
| **예상 방해 횟수** | number | 0~20 | 0 | 예상 중단 횟수 |

#### 퍼스널라이제이션 엔진
| 입력 필드 | 범위 | 기본값 | 설명 |
|---------|------|---------|------|
| **수면 시간(시간)** | 3~12 | 7 | 어제 수면 시간 |
| **컨디션 점수** | 1~5 | 중간 | 오늘 신체 상태 (1=매우피곤, 5=최고조) |
| **외부 일정 강도** | 여유/보통/바쁨 | 보통 | 일정 부하도 |

**출력 정보**:
- 권장 집중 시간
- 예상 방해 허용치
- 세로 루틴 (준비 → 딥워크 → 정리)
- 컨디션 누적 패턴 분석

### 7. 예측 히스토리 관리
#### 데이터 저장 구조

| 항목 | 값 | 설명 |
|------|-----|------|
| **저장 키** | `pomodoro_prediction_history_v1` | localStorage 키 |
| **최대 기록** | 50개 | 초과 시 자동 삭제 |
| **데이터 포맷** | JSON 배열 | 각 항목은 객체 |

#### 예측 기록 테이블 컬럼

| 컬럼 | 데이터 타입 | 설명 |
|------|-----------|------|
| 작업명 | string | 작업 이름 |
| 카테고리 | string | 작업 분류 |
| 예측 시간(분) | number | 예상 소요 시간 |
| 실제 시간(분) | number | 실제 소요 시간 |
| 차이(분) | number | 실제 - 예측 |
| AI 분석 | string | Copilot 분석 결과 또는 "규칙" |

#### 카테고리별 통계

| 통계항목 | 계산식 | 표시 형식 |
|---------|--------|---------|
| 회수 | COUNT(category) | "회" |
| 평균오차 | AVG(실제 - 예측) | "-19.0분" |
| 정확도 | ((예측값-|오차|)/예측값)*100 | "24%" |

### 8. AI 분석 기능 (Copilot Integration)
#### GitHub 토큰 관리
- **토큰 저장**: 브라우저 localStorage에만 저장
- **토큰 키**: `github_copilot_token_v1`
- **보안**: 서버에 전송되지 않음
- **토큰 생성**: GitHub Personal Access Token 필요

#### AI 분석 옵션
- **체크박스**: "Copilot SDK AI 분석 사용"
- **역할**: 예측 정확도 분석 및 피드백 제공
- **분석 항목**: 작업 패턴, 시간 추정 편향, 개선 제안

## 🔧 JavaScript 주요 함수

### 타이머 제어
- `startTimer()` - 타이머 시작
- `pauseTimer()` - 타이머 일시정지
- `render()` - UI 업데이트 및 CSS 변수 설정
- `resetCurrentModeToFull()` - 현재 모드 초기화
- `switchMode(newMode)` - 모드 변경

### 상태 관리
- `loadPredictionHistory()` - 예측 히스토리 로드
- `savePredictionHistory(history)` - 히스토리 저장
- `updateAuthStatus()` - 인증 상태 업데이트
- `loadAuthState()` - 저장된 인증 정보 로드

### AI 통합
- `analyzePredictionWithCopilot(...)` - Copilot SDK 호출
- `generateRoutineWithCopilot(...)` - 루틴 자동 생성
